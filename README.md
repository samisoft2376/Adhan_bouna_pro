# 🕌 أذان بونة Pro | Adhan Bouna Pro
**الجيل الجديد من تطبيقات مواقيت الصلاة - دقة برمجية وتصميم ملكي عصري.**

---

## 🌟 رؤية المشروع
تطبيق **أذان بونة Pro** ليس مجرد صفحة لمواقيت الصلاة، بل هو تجربة رقمية فاخرة صُممت خصيصاً لتجمع بين الروحانية وأحدث تقنيات الويب (PWA). يهدف التطبيق لخدمة المصلين في مدينة **عنابة** وكامل التراب الوطني بدقة متناهية وبواجهة بصرية مريحة للأعصاب.

## ✨ المميزات الحصرية (النسخة Pro)
* 🟢 **الواجهة الملكية (Premium UI):** تصميم يعتمد على اللون الأخضر العميق (Teal) والتدرجات الذهبية الفاخرة التي تليق بالتطبيقات الإسلامية الحديثة.
* 🌌 **إعدادات السماء الليلية (Nighttime Sky Mode):** نافذة إعدادات مبتكرة بخلفية زجاجية شفافة (Glassmorphism) تحاكي جمال السماء والنجوم.
* ⏳ **العداد التنازلي الذكي:** نظام حسابي دقيق يخبرك بالوقت المتبقي للصلاة القادمة بالثانية.
* 📿 **شريط الأذكار التفاعلي:** أذكار وأدعية نبوية تتغير تلقائياً كل 10 ثوانٍ لتعطير لسانك بذكر الله.
* 🔔 **تنبيهات الأذان:** دعم الأذان الصوتي مع إمكانية الاختيار بين أصوات الحرمين الشريفين والأقصى المبارك.
* 📱 **تثبيت كـ App:** بفضل تقنية PWA، يمكنك تثبيت التطبيق على هاتفك والحصول على أيقونة "K" الذهبية في قائمة تطبيقاتك.
* 📶 **العمل بدون إنترنت:** يدعم ميزة التخزين المؤقت (Service Worker) لضمان فتح التطبيق في أي وقت.

---

## 🛠️ البناء التقني (Tech Stack)
تم تطوير التطبيق باستخدام أفضل الممارسات لضمان السرعة والخفة:
- **Frontend:** HTML5, CSS3 (Modern Grid & Flexbox).
- **Logic:** Vanilla JavaScript (ES6+) لإدارة التوقيت والعمليات الحسابية.
- **PWA Core:** Web Manifest & Service Workers لضمان تجربة التطبيق الحقيقي.
- **API:** التكامل مع (Aladhan API) للحصول على مواقيت معتمدة عالمياً.
- **Hosting:** تم النشر عبر GitHub Pages مع دعم النشر التلقائي.
